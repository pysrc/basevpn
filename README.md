# 非常简单的VPN实现

relay用于打洞或中继


# 客户端-服务端通信格式

```

1. 心跳包结构
---------------
|src|TYPE_HART|
---------------

2. 心跳响应包结构
----------------
|src|TYPE_HARTB|
----------------

3. 重置信息包结构

----------
|TYPE_RST|
----------

4. 信息包结构

--------------------------------
|info|src|info length|TYPE_INFO|
--------------------------------

5. ipv4包

------------------------------------------
|ip packet|dst|ip packet length|TYPE_IPV4|
------------------------------------------

```
